# ringle

# 개요
링글 Tech팀 Backend 과제

## 개발 기간
<br> 2025.04.25 - 2025.04.27

## 기능
1. 튜터가 이용할 API
   1) 수업 가능한 시간대 생성 & 삭제
2. 학생이 이용할 API
   1) 기간 & 수업 길이로 현재 수업 가능한 시간대를 조회
   2) 시간대 & 수업 길이로 수업 가능한 튜터 조회
   3) 시간대, 수업길이, 튜터로 새로운 수업 신청
   4) 신청한 수업 조회
